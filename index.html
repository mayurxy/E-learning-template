Learning about Android!


Android is Linux based open source operating system that runs on mobile phones and tablets. Android is based on the Linux kernel and as such is open source so it is able to be manipulated. Since software to write these Android applications is open source so it is free to setup. The latest version is Android 6 Marshmallow, which will gradually become available for more and more devices over the coming months. This is completely opposite to other software like Apple’s IOS or Microsoft’s Windows operating system.







Features of android:


  • Good GUI :

    • Android OS provides good base screen interface for the developer.

  • Database :

    • Uses SQLite lightweight relational database can be used for data storage

  • Connectivity:

    • wi-fi, Bluetooth and many new available technologies like NFC are supported by android

  • Multitouch and multitask:

    • One can switch between multiple apps easily.

    And many more…… Android applications are generally developed in JAVA language using Android software development kit. Those apps are hosted on Google Play Store, Amazon Appstore and other locations. Android is growing fast and has been largest installed base mobile platform in the world. This module aims to learn the architecture of android, environment setup to make simple android app.

    Architecture of Android


    The Linux Kernel Positioned at the bottom of the Android software stack, the Linux Kernel provides a level of abstraction between the device hardware and the upper layers of the Android software stack. The kernel provides preemptive multitasking, low-level core system services such as memory, process and power management in addition to providing a network stack and device drivers for hardware such as the device display, Wi-Fi and audio.

    Preparing environment


    OS requirements:

    User should have one of the following operating systems to develop android applications:

  • Microsoft Windows XP or later version.
  • Linux including GNU C Library or later (Ubuntu).
  • Mac OS X 10.5.8 or later version with Intel chip.

    • Other required tools to create the environment to develop android applications which are freely available.
    • Java JDK

      • Get the latest version for respective operating system and architecture from Oracle website: Link to JAVA JDK.

    • Setting up PATH

      • After installing java development kit we need to configure the setup by setting up PATH and JAVA_HOME variable. This refers to the directory that contains java and javac. Right-click on My Computer, select Properties, then Advanced, then Environment Variables. Then, you would update the PATH value.

    • Android SDk

      • Android SDK can be installed as IDE package or plug-in or command line tools can be listed in : Link to Android SDK.

    • Java runtime environment JRE

      • Get the latest version for respective operating system and architecture from Oracle website:Link to JRE download.

    • Android Studio or other IDE

      • Official IDE to develop the android applications can be downloaded from:Eclipse Downloads. Alternatively, Eclipse IDE can be used for coding the apps using installing Android plug-in for same IDE which is available at: Link to plug-in. Here installing the android package for eclipse IDE can be sophisticated. Follow the link for small tutorial to install these packages: Plug-in in stalltion tutorial.
